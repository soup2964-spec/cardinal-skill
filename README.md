# Cardinal Skill

Cursor Agent Skill for building UI that matches [trycardinal.ai](https://trycardinal.ai/).

## Install

Copy this folder into your Cursor skills directory:

```
~/.cursor/skills/cardinal-design-system/
```

Or clone into that path:

```bash
git clone https://github.com/soup2964-spec/cardinal-skill.git ~/.cursor/skills/cardinal-design-system
```

## Files

| File | Purpose |
|------|---------|
| `SKILL.md` | Main skill — tokens, typography, workflow, checklist |
| `reference.md` | Component specs, assets, colors |
| `examples.md` | Section templates and prompt examples |

## Usage

In Cursor chat:

```
Use the cardinal-design-system skill to add a features section
```

The agent auto-applies this skill when you mention Cardinal, trycardinal.ai, or cardinal-replica UI work.
